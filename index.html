Mengenal Version Control System, Git, dan GitHub

1. Mengenal GitHub

  • Membuat akun dan repository di github.com
      • - Nama repository by dedehsh (contoh: codingmum-belajar-git)
      - Referensi: https://www.youtube.com/watch?v=Q3Id0DgcrXY
  • Melakukan commit untuk menambah/merubah file
      • - Buat file baru di repository, index.html
      - Isi dari index.html bisa buat sendiri atau copy paste dari: https://github.com/wnitya2/codingmum_training/blob/master/index.html
      - Edit isi dari index.html (coba-coba saja edit isinya)
  • Mempublish website sendiri di GitHub Pages
      • Referensi: https://www.youtube.com/watch?v=BA_c3bGQXlQ
  • Menginfokan link github dan website yang telah di publish di Group KCMI Newbie
  • Have fun! Mari bermain-main di GitHub
